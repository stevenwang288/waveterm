# docs/内部记录（索引）

本目录用于存放**非 Docusaurus 官网文档**的内部记录（需求、阶段状态、差异对比、待办与验收等），避免 `docs/` 根目录被零散文件污染。

## 目录结构

- `WAVE改造记录_2026-02-22.md`：从开始到现在对 WAVE/WaveCN 的改造总记录（详细版）。
- `WAVE改造复现指南_2026-02-22.md`：从官方仓库基线恢复到当前交付状态的可复现操作指南。
- `阶段状态/`：按日期的阶段性收口/修复记录。
- `差异对比/`：相对上游 `wavetermdev/waveterm` 的差异梳理。
- `待办/`：用户需求清单与 backlog。
- `构建与验证/`：Windows/Linux 构建、打包与回归验证流程。
- `需求/`：语音播报等关键功能的需求定义与验收标准。

## 说明

- `codex日志/` 为本地/一次性产物（已在本目录 `.gitignore` 中忽略），仅用于回看排查，不进入版本库。
